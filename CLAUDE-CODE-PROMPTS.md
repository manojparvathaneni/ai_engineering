# Claude Code Starter Prompts

## Initial Setup
```
I'm working on my AI Engineering course repository. Please read CLAUDE.md for context.
The repo is at: [your path]/ai-engineering
What would you like to work on?
```

## Starting a New Week
```
I'm starting Week [X]: [Topic]. Please help me:
1. Set up the folder structure
2. Create the README
3. Set up project venv
Topics: [paste syllabus]
```

## Creating Teaching Materials
```
I finished learning [TOPIC]. Create:
1. Teaching guide (intuition first, analogies)
2. Runnable demo
Put in: week-XX/teaching-materials/
```

## Building Project Feature
```
Working on Project [X]. Current status: [describe]
Need to add: [feature]
Follow conventions in CLAUDE.md.
```

## Research Task
```
I encountered [PAPER/CONCEPT]. Help me:
1. Summarize key ideas
2. Relate to course
3. Create notes
4. Teaching angle
```

## Python Deep Dive
```
I encountered [PYTHON CONCEPT] in the course and want to understand it deeply.

Please create in python-learning/concepts/[concept-name]/:
1. README.md - explanation with examples
2. examples.py - working code examples with inline deps
3. exercises.py - practice problems

Focus on:
- Why this exists (the problem it solves)
- How it works (mechanics)
- Where I'll see it (real-world usage)
```

## Python Practice
```
Create practice exercises for [TOPIC] in python-learning/exercises/
Include varying difficulty levels and test cases.
```

## End of Week Summary
```
Completed Week [X]. Help me:
1. Summarize what was covered
2. List materials created
3. Update README
4. Prep for next week
```
